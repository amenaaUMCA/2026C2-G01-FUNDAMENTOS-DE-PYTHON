3. La ruta mental de un programa confiable

1LimpiarEliminar espacios y normalizar texto.
2ConvertirPasar de str a int o float.
3ValidarComprobar que los datos tengan sentido.
4DecidirAplicar las reglas con condiciones.
5MostrarComunicar el resultado claramente.
Idea clave: limpiar no es lo mismo que validar. Limpiar mejora el formato; validar determina si el dato puede utilizarse.

Sin limpiar

respuesta = input("¿Acepta? ")

    print("Aceptado")

" Si " no coincide con "si".

Entrada normalizada

respuesta = input("¿Acepta? ")
respuesta = respuesta.strip().lower()

if respuesta == "si":
    print("Aceptado")

Distintas formas de escribir se comparan de manera consistente.

5. Extracción y búsqueda

Índice texto[0] obtiene el primer carácter.
Slicing texto[0:2] obtiene una sección.
Búsqueda "@" in correo comprueba existencia. 
codigo = "PY-101-2026"
lenguaje = codigo[:2]
anio = codigo[-4:]

print(f"Lenguaje: {lenguaje} | Año: {anio}")

6. Entrada y conversión de datos

Regla de oro: todo lo recibido mediante input() es texto, aunque el usuario escriba números.

7. Validación segura con try-except

try-except permite responder de forma controlada cuando una conversión no puede realizarse.

try:
    edad = int(input("Edad: "))

    if edad < 0:
        print("La edad no puede ser negativa")
    else:
        print(f"Edad registrada: {edad}")
except ValueError:
    print("Debe ingresar un número entero")
Orden recomendado: primero convertir, después validar el rango y finalmente aplicar las reglas del problema.

Validación de campos vacíos

nombre = input("Nombre: ").strip()

if not nombre:
    print("El nombre es obligatorio")
else:
    print(f"Bienvenido, {nombre}")

8. Código limpio y PEP 8

Difícil de leer

n="Ana"
p=500
if n=="Ana":p=0
print("Total:"+str(p))

Claro y mantenible

nombre_cliente = "Ana"
total_pagar = 500

if nombre_cliente == "Ana":
    total_pagar = 0

print(f"Total: {total_pagar}")
  • Use nombres descriptivos en snake_case.
  • Use cuatro espacios para la indentación.
  • Deje una instrucción por línea.
  • Use constantes en mayúscula: IMPUESTO_IVA = 0.13.
  • Prefiera f-strings para construir mensajes.
